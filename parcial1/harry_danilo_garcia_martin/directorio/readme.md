Format name:
	server_#.txt	'server' in lowcase, '#' number id, 'txt' file extension.
each row format:
Function|'server_funtion'
Brand|'server brand or manufacturer'
Frec_processor[MHz]|
Cant_processor[#]|
Cores[#]|
Ram[Gb]|
Cant_Hdd[#]|
Tot_cap_Hdd[Gb]|
Pot[Watts]|
Location|'server_location'
Example:

File name=server_32.txt
into server_32.txt:

Function|treasury_files
Brand|compaq
Frec_processor[MHz]|3200
Cant_processor[#]|1
Cores[#]|4
Ram[Gb]|6
Cant_Hdd[#]|2
Tot_cap_Hdd[Gb]|1400
Pot[Watts]|650
Location|ed_A_reception
END
